# Workflow

GenAI Span Mapper is a paid remote MCP schema mapper for OpenTelemetry GenAI spans, provider attributes, missing fields, dashboard schemas, and mapping receipts.

## Repeatable Flow

1. Submit span samples, provider names, and field dictionaries.
2. Map fields into OpenTelemetry GenAI attributes and dashboard schemas.
3. Flag missing attributes and provider-version drift.
4. Return normalized JSON and archive a mapping receipt.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
