# Quickstart

GenAI Span Mapper MCP is designed for observability teams, LLM platform teams, data engineers, and SREs.

## Fast Path

1. Open the hosted product and choose the default plan.
2. Create or request a bearer token from the product website.
3. Add the Streamable HTTP MCP endpoint to a compatible client.
4. Send a tools/list request first, then call the specific tool with public-safe sample data.
